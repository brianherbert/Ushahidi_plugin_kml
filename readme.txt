== Installation ==
1. Copy the entire /kml/ directory into your /plugins/ directory.
2. Activate the plugin through the administrator Addons interface at /admin/addons/plugins/
3. Navigate to http://www.yourwebsite.com/kml to download the KML (actually a compressed KMZ file)
4. Options for balloon content, caching, etc. are available in variables section at top of /kml/views/kml.php
